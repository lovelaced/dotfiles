# dotfiles

Add any config dirs into dotfile_setup.py, add them to folder, then deploy.

Requires the following to be installed:

* bspwm
* sxhkd
* zsh
* xmodmap
* xbindkeys
* zsh-syntax-highlighting
* weechat

Bar, which requires conky, can be found in the bspwm-conkbar repo.
